# Curriculum Overview: Path to Staff-Level Mastery

## Learning Path (5 Levels, 13 Topics, 55+ Subtopics)

### Level 1: Foundations
*Build the base everything else rests on.*

**CS Fundamentals** (5 subtopics, ~80 cards)
Big O Notation > Binary & Bits > How Computers Work > OS Basics > Boolean Logic

**JavaScript / TypeScript** (7 subtopics, ~125 cards)
JS Execution Model > Types & Coercion > Closures & Scope > Prototypes & Classes > Promises & Async > TS Type System > Advanced TS

**Python Fundamentals** (4 subtopics, ~70 cards)
Data Model > Iterators & Generators > Decorators & Context Managers > Concurrency

---

### Level 2: Core
*The bread and butter of coding interviews.*

**Data Structures** (8 subtopics, ~140 cards)
Arrays & Strings > Linked Lists > Stacks & Queues > Hash Maps > Trees & BSTs > Heaps > Graphs > Tries

**Algorithm Interview Patterns** (8 subtopics, ~155 cards)
Two Pointers > Sliding Window > Binary Search > Recursion & Backtracking > Dynamic Programming > Greedy > Sorting & Searching > Graph Algorithms

---

### Level 3: Intermediate
*Architecture, frameworks, and design thinking.*

**SOLID & Design Patterns** (4 subtopics, ~85 cards)
SOLID Principles > Creational Patterns > Structural Patterns > Behavioral Patterns

**Clean Architecture & DDD** (4 subtopics, ~80 cards)
Clean Architecture > DDD Strategic > DDD Tactical > Hexagonal Architecture

**React, Next.js, Vercel** (8 subtopics, ~160 cards)
React Core > Hooks Deep Dive > React Patterns > Performance > State Management > Next.js App Router > Next.js Data Patterns > Vercel & Deployment

**SQL Mastery** (4 subtopics, ~80 cards)
SQL Fundamentals > Advanced SQL > Query Optimization > Database Design

---

### Level 4: Advanced
*Systems thinking at scale.*

**System Design** (5 subtopics, ~120 cards)
Fundamentals > Components > Database Design at Scale > Case Studies > API Design

**Distributed Systems** (4 subtopics, ~75 cards)
Fundamentals > Patterns > Messaging & Streaming > Observability

**Cryptography & Networking** (4 subtopics, ~75 cards)
Crypto Fundamentals > TLS & Certificates > Networking Protocols > Web Security

---

### Level 5: Expert / Staff-Level
*Deep specialization.*

**Math for AI** (4 subtopics, ~90 cards)
Linear Algebra > Calculus & Optimization > Probability & Statistics > ML Fundamentals

---
